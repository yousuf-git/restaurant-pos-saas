Keyboard Shortcuts (integrated into PosOrders)

| Shortcut      | Action                                                        |
|---------------|---------------------------------------------------------------|
| / or Ctrl+K   | Focus search bar                                              |
| Tab           | Switch focus between item grid and bill panel                 |
| Arrow keys    | Navigate items in grid (Up from top row returns to search)    |
| Enter         | Add focused item to order                                     |
| + / -         | Increment/decrement quantity in bill                          |
| Escape        | Clear search (if in search) or clear all bill items           |
| Ctrl+P        | Reprint last receipt                                          |
| Ctrl+Enter    | Confirm order + print                                         |
